NinjaUI jQuery UI library

Just playing around with NinjaUI for a private project. It’s a nice stylish UI plugin for jQuery. It’s not that large and complex than jQuery UI. It has less feature and options but in my opinion the better UI design. One of the unique features are the symbols realized with webfonts. Take a look at it, but beware it’s still beta.
